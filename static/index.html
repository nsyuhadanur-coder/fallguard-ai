How It Works

01

Capture

Webcam frames are captured at 2 fps and sent to the server via WebSocket for real-time processing.

02

Inference

Each frame is resized to 224×224 and passed through MobileNetV2 trained on fall/non-fall images.

03

Alert

When a fall is detected with high confidence, an immediate visual and audio alert is triggered.

04

Log

Every fall event is timestamped and stored in the alert log for review and monitoring history.
